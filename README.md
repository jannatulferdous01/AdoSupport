# AdoSupport

AdoSupport is a multi-branch project designed to streamline backend and frontend development with modular capabilities. This repository serves as a foundational codebase to build applications with clear separation of concerns, supporting scalable and maintainable architectures.

## Getting Started

1. **Clone the repository:**
   ```sh
   git clone https://github.com/jannatulferdous01/AdoSupport.git
   ```

2. **Checkout your desired branch:**
   ```sh
   git checkout backend      # For backend
   git checkout frontend     # For frontend
   git checkout modular-backend  # For modular backend
   ```

3. **Install dependencies:**  
   Proceed to each branch directory and follow instructions in respective README files (if available).
